# www 🗺️

![PREVIEW](./preview.png)

A modern web application built with blazing-fast technologies.

## Features

- **Language support (RU/EN)**
  - Language selection based on the country you are in (using ipdata)
    - Proxy fetching (hiding secret key in URL's query)
    - Response caching
- **Strong ESLint rules**
- **Data Management**
  - Text, colors, constants content in `data` folder
  - Insertion and conversion of the text into TSX structures
- **Discord Presence**
- **Files Managment**
